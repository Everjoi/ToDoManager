
Опис завдання:
Ваше завдання - розробити простий веб-застосунок ToDo за допомогою 
ASP.NET MVC (NET CORE 8) та мови програмування C#. 
Застосунок повинен мати наступні функціональності:

Список завдань (Tasks):
Відображення списку завдань.
Можливість додавання нового завдання.
Можливість відмічення завдання як виконаного.
Редагування та Видалення:
Можливість редагування назви завдання.
Можливість видалення завдання.

Архітектура:
Використання паттерну MVC для організації коду.
Використання Entity Framework для роботи з базою даних.

Інтерфейс:
Привітний та зручний інтерфейс користувача.

Код:
Чистий, добре організований та документований код.
Використання найкращих практик програмування в мові C# та ASP.NET MVC.

База даних:
Використання Code-First підходу для створення бази даних за допомогою
Entity Framework.

Функціональність:
Всі описані функціональності повинні працювати правильно та ефективно.
Інтерфейс повинен бути зручним для користувача.

Безпека:
Захист від потенційних атак та зловживань.

Додаткові завдання (бонус):
Реалізація функції сортування або фільтрації завдань.
Реалізація AJAX-запитів для покращення реактивності інтерфейсу.
